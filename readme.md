# Dotfiles
These are configuration files for my operating system. They use [flexoki colorscheme](https://stephango.com/flexoki).

Required software:
- i3,
- alacritty,
- fish,
- fastfetch,
- neovim (along with some other dependencies like xclip etc.),
- ranger,
- zathura,
- flameshot,
- ly,
- polybar,
- picom,
- pamixer.
